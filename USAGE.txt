The code should run autonomously after flashing the devices as long as power is supplied to the two ESP32-H2s and the Teensy. The current audio input sample is the "Hi, I'm talking from over here" audio from lab. You can hear the output by connecting a headphone to the output Teensy audio shield jack. It is important that the serial connections between the receiver ESP32-H2/antenna array and the Teensy match the UART ports specified in the code as without this the Teensy cannot receive the data and thus no output is produced. Opening Serial Monitor for the two ESP32-H2s and the Teensy shows debugging outputs on transmission, reception, and processing of the audio signal.
