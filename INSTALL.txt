To compile the receiver_tag code, I used the ESP IDF extension in VSCode. I used the Windows version rather than the WSL connection as WSL cannot sense COM ports without going through a process with the usbipd-win tool to add the COM ports to WSL. I flashed the code to the ESP32-H2 by setting the target to an ESP32-H2, specified the COM port as to which the ESP32-H2 was connected via USB-C, and built and flashed the code using the ESP IDF interface. After flashing, the code should run immediately on the ESP32-H2 and continue to run as long as there is power.
